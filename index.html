Jquery.responsivefonts

JQuery plugin which makes fonts responsive

Download .zip Download .tar.gz View on GitHub

Include JavaScript and CSS files, and then just call the plugin on the selector you want.

Note that plugin only changes body font size, so all other sizes should be defined using ems.

$('.row').responsiveFonts({
    minWidth : 960,
    maxWidth : 1920
});

With min and max width parameters you decide when the plugin should stop resizing fonts.
